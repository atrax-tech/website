# Основные определения

Философия из всех наук «самая фундаментальная». Поэтому и начнем мы с точного значения слов, то есть — с понятийного аппарата.

## Информация
Информация это структурированные данные о мире. Мир состоит из единичных фактов, информация — из единичных данных об этих фактах. Данные, описывающие объект, по отдельности являются его свойствами, а вместе — его состоянием.

## Исполнитель
Исполнитель это объект, способный выполнять действия. Выполнение действий может менять свойства исполнителя, тем самым переводя его из одного состояния в другое. Все возможные состояния и переходы между ними составляют граф состояний. 

## Алгоритм

Алгоритм — это последовательность действий, выполняемых исполнителем для решения задачи. В графе состояний это маршрут, двигаясь по которому, исполнитель достигает цели.

## Параллелизм

Может показаться, что слово «последовательность» в определении алгоритма некорректно из-за существования параллельных вычислений. Но нет, слово правильное — просто по графу состояний могут «бродить» несколько исполнителей, выполняющих собственные алгоритмы.

## Программа

Команда — это понятный исполнителю сигнал для выполнения действия. Полный набор таких команд называется языком исполнителя. Запись алгоритма на языке исполнителя называется программой.
